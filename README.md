# Amazon Sales Dashboard (Excel Project)

## Overview
This project analyzes Amazon sales data using Microsoft Excel to uncover key business insights.

## Tools Used
- Microsoft Excel
- Pivot Tables
- Data Cleaning
- Data Visualization

## Key Insights
- Set category generates the highest revenue
- Maharashtra is the top-performing state
- Sales peak during April–June indicating seasonal demand
